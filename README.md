# jira-importer
Importer for Jira issues.

## Console Application Usage
### Mandatory Arguments
* `--jira` or `-j` - The URL of the Jira.
* `--username` or `-u` - The username to connect to the Jira.
* `--password` or `-p` - The password of the user.

Example:
* `-j "http://jira.com" -u "joe" -p "password"`
